# Dryft User Onboarding Guide

This document is a user-facing overview of getting started, verification, and safety features.

## 1. Getting Started

1. Create an account with email and password.
2. Set your display name and profile photo.
3. Complete onboarding prompts to set preferences.

## 2. Verification Process

Dryft requires verification for access to core features:

- **Age gate**: card or ID verification.
- **Identity checks**: selfie/photo verification and optional social verification.

Verification helps keep the platform safe and restricted to adults.

## 3. Matching & Chat

- Swipe in the Discovery feed to like or pass.
- When both users like each other, a match is created.
- Start chatting in the Conversations tab.

## 4. VR Companion Sessions

- Join a session using a 6-digit code or invite link.
- View real-time VR activity and chat in session.
- Optional haptic interactions are available with permission.

## 5. Safety Features

- **Block & Report**: instantly block a user or report misconduct.
- **Panic Button**: immediately exit unsafe interactions.
- **Safety Warnings**: moderation warnings are delivered in-app.

## 6. Account & Privacy Controls

- Manage profile visibility and preferences in Settings.
- Update notification preferences at any time.

## Need Help?

Contact support at `support@dryft.site`.
